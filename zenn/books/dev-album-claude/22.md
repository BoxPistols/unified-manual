# 
            ウィンドウとペインの操作
          

          
            画面を自由自在に分割・整理して、マルチタスク環境を構築します。
          

        


        
          
            ## 
              ウィンドウ操作 (Tabs)
            
            
              iTerm2の「タブ」に相当します。全画面を切り替えて使いたい場合に便利です。
            


            
              
                
                  作成

                  `Prefix + c`
                

                新しいウィンドウを開きます。

              

              
                
                  名前変更

                  `Prefix + ,`
                

                ウィンドウの名前を変更します。

              

              
                
                  次のウィンドウ

                  `Prefix + n`
                

                番号順に次のウィンドウへ移動。

              

              
                
                  前のウィンドウ

                  `Prefix + p`
                

                番号順に前のウィンドウへ移動。

              

              
                
                  番号指定ジャンプ

                  `Prefix + 0..9`
                

                指定した番号のウィンドウに直接移動します。

              

            

          


          
            ## 
              ペイン操作 (Splits)
            
            
              1つの画面を分割して使います。ログを見ながらコマンドを打つ時などに最適です。
            


            
              
                ### 画面分割
                
                  
                    
                      

                      

                    

                    左右分割

                    `Prefix + %`
                  

                  
                    
                      

                      

                    

                    上下分割

                    `Prefix + "`
                  

                

              


              
                ### 移動と調整
                
                  - 
                    ペイン間の移動

                    `Prefix + 矢印キー`
                  
                  - 
                    ペインのズーム（最大化/戻す）

                    `Prefix + z`
                  
                  - 
                    ペインを閉じる

                    `Ctrl + d (または exit)`
                  
                  - 
                    ペイン番号の表示

                    `Prefix + q`
                  
                

              

            

          


          
            ## 
              AIコーディング実践配置
            

            :::message
**推奨レイアウト**
AIコーディングを行う際の個人的なおすすめレイアウトです。
              
                まず Prefix + % で左右に分割
                右側のペインで Prefix + " で上下に分割
                左側 (大): Claude Code CLI / エディタ
                右上 (小): サーバーログ / 監視
                右下 (小): コマンド実行 / テスト
:::
          


          
             ## 
              まとめ
            
            
              これでtmuxの基本的な操作はマスターしました。ウィンドウで作業コンテキストを分け、ペインで情報を一覧化することで、作業効率は劇的に向上します。
            

          

        


        ### チャレンジ: ウィンドウとペインのコマンドを書いてみよう

tmux コマンドラインから、ウィンドウの作成・名前変更、ペインの分割を行うコマンドを書いてください。

```tsx
# ウィンドウとペインの操作コマンド
# 1. "editor" という名前の新しいウィンドウを作成

# 2. 現在のペインを左右に分割

# 3. 現在のペインを上下に分割

```

:::details 模範解答
```tsx
# ウィンドウとペインの操作コマンド
# 1. "editor" という名前の新しいウィンドウを作成
tmux new-window -n editor

# 2. 現在のペインを左右に分割
tmux split-window -h

# 3. 現在のペインを上下に分割
tmux split-window -v
```
:::
