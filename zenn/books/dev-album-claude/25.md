# 
            プラグイン導入
          

          
            Tmux Plugin Manager (TPM) を使って、機能を拡張しましょう。
          

        


        
          
            ## 
              TPM (Tmux Plugin Manager) とは
            
            
              tmuxのプラグインを簡単に管理するためのツールです。これを入れるだけで、テーマの変更やセッション保存などの高度な機能が簡単に使えるようになります。
            

          


          
            ## 
              導入ステップ
            

            
              
                ### 
                  1. TPMのインストール
                
                
                  GitHubからTPMをクローンしてきます。
                

                ```bash
$ git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
              


              
                ### 
                  2. tmux.confへの記述
                
                
                  `~/.tmux.conf` の一番下に以下の行を追加します。
                

                ```bash
# プラグインのリスト
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'

# TPMの初期化（必ずファイルの最後に記述！）
run '~/.tmux/plugins/tpm/tpm'
```
              


              
                ### 
                  3. 設定の反映とインストール
                
                
                  以下の手順でプラグインを有効化します。
                

                
                  - tmuxを起動（または設定を再読み込み `tmux source ~/.tmux.conf`）
                  - **Prefix + I** (大文字のアイ) を押す
                  - "TMUX environment reloaded." と表示されれば完了です
                

              

            

          


          
            ## 
              おすすめプラグイン: Dracula テーマ
            
            
              見た目を一瞬でかっこよくするテーマプラグインを導入してみましょう。
            

            
            
              ### 設定方法
              
                `~/.tmux.conf` のプラグインリストに以下を追加します。
              

              ```bash
set -g @plugin 'dracula/tmux'

# オプション設定（日付、時間、天気を表示など）
set -g @dracula-plugins "battery cpu-usage ram-usage time"
set -g @dracula-show-powerline true
set -g @dracula-show-flags true
set -g @dracula-show-left-icon session
```
              
                追加後、再び **Prefix + I** を押してインストールすれば、ステータスバーが劇的に変化します。

          ### チャレンジ: TPM プラグイン設定を書こう

~/.tmux.conf に TPM（Tmux Plugin Manager）とプラグインの設定を書いてください。TPM 本体、tmux-sensible、Dracula テーマを含めましょう。

```tsx
# ~/.tmux.conf プラグイン設定

# プラグインリスト
# TPM 本体

# tmux-sensible（推奨設定セット）

# Dracula テーマ

# TPM の初期化（ファイルの最後に記述）
```

:::details 模範解答
```tsx
# ~/.tmux.conf プラグイン設定

# プラグインリスト
# TPM 本体
set -g @plugin 'tmux-plugins/tpm'

# tmux-sensible（推奨設定セット）
set -g @plugin 'tmux-plugins/tmux-sensible'

# Dracula テーマ
set -g @plugin 'dracula/tmux'
set -g @dracula-plugins "battery cpu-usage ram-usage time"

# TPM の初期化（ファイルの最後に記述）
run '~/.tmux/plugins/tpm/tpm'
```
:::
