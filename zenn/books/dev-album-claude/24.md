# 
            生産性向上の設定
          

          
            より直感的で高速な操作を実現するためのキーバインディングを追加します。
          

        


        
          
            ## 
              キーバインドの変更方法
            
            
              `bind` コマンドを使って、キーの割り当てを変更できます。デフォルトの使いにくいキーを、覚えやすいキーに変更しましょう。
            

          


          
            ## 
              必須カスタマイズ3選
            

            
              
                ### 
                  1. プリフィックスキーの変更（オプション）
                
                
                  Ctrl+B は指が届きにくい場合があります。Caps LockをCtrlに変更しているユーザーには **Ctrl+A** が人気です。
                

                ```bash
# PrefixをCtrl+aに変更
set -g prefix C-a
unbind C-b
bind C-a send-prefix
```
              


              
                ### 
                  2. 画面分割を直感的にする
                
                
                  デフォルトの `%` と `"` は覚えにくいです。パイプ `|` で左右、ハイフン `-` で上下に分割するように変更します。
                

                ```bash
# | でペインを縦に分割する
bind | split-window -h

# - でペインを横に分割する
bind - split-window -v
```
              


              
                ### 
                  3. 設定ファイルの再読み込みを簡単にする
                
                
                  設定を変更するたびにコマンドを打つのは面倒です。`Prefix + r` で再読み込みできるようにします。
                

                ```bash
# 設定ファイルをリロードする
bind r source-file ~/.tmux.conf \; display "Reloaded!"
```
              

            

          


          
            ## 
              Vimユーザー向け設定
            
            
              Vimを使用するエンジニアやエンジニアには、ペイン移動もVimライクにすることをお勧めします。
            

            ```bash
# Vimのキーバインドでペインを移動する
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R
```
          


          
             ## 
              まとめ
            
            
              ここまでの設定を追加した `~/.tmux.conf` の例です。
            

            ```bash
# ---------------------------
# 基本設定
# ---------------------------
set -g mouse on
set -g base-index 1
setw -g pane-base-index 1
set -g default-terminal "screen-256color"

# ---------------------------
# キーバインド
# ---------------------------
# | でペインを縦に分割する
bind | split-window -h
# - でペインを横に分割する
bind - split-window -v

# Vimのキーバインドでペインを移動する
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# 設定ファイルをリロードする
bind r source-file ~/.tmux.conf \; display "Reloaded!"
```
            :::message
**完了**
このファイルを保存し、tmux source ~/.tmux.conf を実行（または tmux を再起動）すれば、新しいショートカットが使えるようになります。
:::
          

          ### チャレンジ: tmux.conf のキーバインドを設定しよう

~/.tmux.conf に追加する生産性向上のキーバインド設定を書いてください。ペイン分割の直感的なキーと設定リロードを含めましょう。

```tsx
# ~/.tmux.conf キーバインド設定

# 1. | で左右にペイン分割

# 2. - で上下にペイン分割

# 3. Vim風のペイン移動 (h/j/k/l)

# 4. Prefix + r で設定リロード
```

:::details 模範解答
```tsx
# ~/.tmux.conf キーバインド設定

# 1. | で左右にペイン分割
bind | split-window -h

# 2. - で上下にペイン分割
bind - split-window -v

# 3. Vim風のペイン移動 (h/j/k/l)
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# 4. Prefix + r で設定リロード
bind r source-file ~/.tmux.conf \; display "Reloaded!"
```
:::
