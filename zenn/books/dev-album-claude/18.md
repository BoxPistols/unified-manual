# 
            インストール確認
          

          
            tmuxが正しくインストールされたことを確認します。
          

        


        
          
            ## バージョン確認
            
              まず、tmuxが正しくインストールされているか、バージョンを確認します。
            

            
              ```bash
$ tmux -V
tmux 3.3a
```
              
                バージョン番号が表示されれば、インストールは成功です。
              

            

          


          
            ## 最初のセッション作成
            
              実際にtmuxセッションを作成して、動作を確認します。
            

            
              
                ### 
                  ステップ 1: 新しいセッションを作成
                
                ```bash
$ tmux new-session -s test
```
                
                  このコマンドを実行すると、tmuxセッションが起動します。ターミナルの下部に緑色のステータスバーが表示されます。
                

              


              
                ### 
                  ステップ 2: セッションから抜ける
                
                
                  tmuxセッションから抜けるには、以下のキーボードショートカットを使用します。
                

                ```bash
# Ctrl+B を押してから D を押す
Ctrl+B D
```
              


              
                ### 
                  ステップ 3: セッション一覧を確認
                
                ```bash
$ tmux list-sessions
test: 1 windows (created Fri Feb 21 12:00:00 2026)
```
              

            

          


          
            ## 次のステップ
            
              ここまでで、tmuxの基本的な環境準備が完了しました。次のセクションでは、tmuxの基本操作を詳しく学びます。
            

            :::message
**準備完了**
tmuxが正常に動作することが確認できたら、次のセクション「tmux基礎」に進みましょう。
:::
          

          ### チャレンジ: tmux セッションの基本操作を練習しよう

tmux セッションの作成、確認、デタッチ、再接続、終了の一連の操作を書いてください。

```tsx
# tmux の基本操作

# 1. test という名前のセッションを作成

# 2. セッションからデタッチ

# 3. セッション一覧を確認

# 4. セッションに再接続

# 5. セッションを終了
```

:::details 模範解答
```tsx
# tmux の基本操作

# 1. test という名前のセッションを作成
tmux new-session -s test

# 2. セッションからデタッチ
Ctrl+B D

# 3. セッション一覧を確認
tmux list-sessions

# 4. セッションに再接続
tmux attach -t test

# 5. セッションを終了
tmux kill-session -t test
```
:::
