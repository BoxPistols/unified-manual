```tsx
# 新規作成:
# 次へ移動:

```

:::details 模範解答
```tsx
# tmux 主要操作の復習

# セッション操作
# 名前付きセッション作成:
tmux new -s dev
# セッション一覧:
tmux ls
# セッション再接続:
tmux attach -t dev

# ウィンドウ操作（Prefix +）
# 新規作成: Ctrl+B C
# 次へ移動: Ctrl+B N
# 名前変更: Ctrl+B ,

# ペイン操作（Prefix +）
# 左右分割: Ctrl+B %
# 上下分割: Ctrl+B "
# ズーム切替: Ctrl+B Z
```
:::
