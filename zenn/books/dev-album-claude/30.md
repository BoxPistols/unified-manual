# 
            トラブルシューティング
          

          
            よくある問題とその解決策、そしてこれからの学習リソース。
          

        


        
          
            ## 
              よくある問題 (FAQ)
            

            
              
                ### Q. Vimの色がおかしい（白黒になる）
                
                  **A.** ターミナルタイプの設定が正しくありません。`~/.tmux.conf` に以下が記述されているか確認してください。
                

                ```bash
set -g default-terminal "screen-256color"
```
              


              
                ### Q. マウスでスクロールできない
                
                  **A.** マウスモードが無効になっている可能性があります。また、コピーモードに入る必要がある場合もあります。
                

                ```bash
# 設定ファイルに追加
set -g mouse on
```
                
                  または、`Prefix + [` でコピーモードに入ると、矢印キーやマウスでスクロールできるようになります。終了するには `q` を押します。
                

              


              
                ### Q. tmuxがフリーズして動かない
                
                  **A.** 特定のペインだけ固まった場合は `Ctrl+C` を試してください。tmux全体がおかしい場合は、別のターミナルタブを開いて、強制終了できます。
                

                ```bash
# 全てのtmuxプロセスを終了する（注意：作業内容は消えます）
$ tmux kill-server
```
              


              
                ### Q. プレフィックスキーが効かない
                
                  **A.** Caps Lockなどが干渉しているか、設定ファイルの記述ミスが考えられます。設定をリロードしても直らない場合は、一度tmuxを完全に終了（`exit` または `tmux kill-server`）して再起動してください。

          ### チャレンジ: tmux のトラブルシューティングを練習しよう

よくある tmux の問題に対する解決コマンドを書いてください。色の問題、マウス有効化、tmux のリセットを含めましょう。

```tsx
# tmux トラブルシューティング

# 1. 256色対応の設定

# 2. マウスモードを有効化

# 3. tmux サーバーを強制終了

# 4. 設定ファイルをリロード
```

:::details 模範解答
```tsx
# tmux トラブルシューティング

# 1. 256色対応の設定（~/.tmux.conf に追加）
set -g default-terminal "screen-256color"

# 2. マウスモードを有効化（~/.tmux.conf に追加）
set -g mouse on

# 3. tmux サーバーを強制終了（全セッション消去）
tmux kill-server

# 4. 設定ファイルをリロード
tmux source ~/.tmux.conf
```
:::
