# 
            前提知識
          

          
            tmuxを学ぶ前に、ターミナルの基本的な操作を理解しておくことが重要です。
          

        


        
          
            ## 
              必要な前提知識
            

            
              このマニュアルを進める前に、以下の知識があると学習がスムーズです。
            


            
              
                ### ✓ ターミナルの基本操作
                
                  cd、ls、mkdir、rm などの基本コマンドを使ったことがある
                

              


              
                ### ✓ ファイルエディタの使用経験
                
                  vim、nano、またはその他のテキストエディタでファイルを編集したことがある
                

              


              
                ### ✓ 環境変数の概念
                
                  PATH、HOME などの基本的な環境変数について理解している
                

              


              
                ### ✓ プロセスの概念
                
                  バックグラウンドプロセス、フォアグラウンドプロセスの違いを理解している
                

              

            

          


          
            ## 
              初心者向けのヒント
            

            :::message
**心配不要です**
上記の知識がない場合でも、このマニュアルを進めることはできます。わからない用語が出てきたら、その都度調べながら進めてください。学習の過程で自然と身につきます。
:::
          


          
            ## 
              ターミナルの動作確認
            

            
              以下のコマンドを実行して、ターミナルが正しく動作していることを確認してください。
            


            
              
                ### 
                  基本的なコマンド確認
                
                ```bash
# 現在のディレクトリを表示
$ pwd
/Users/username

# ホームディレクトリのファイル一覧を表示
$ ls ~
Desktop    Documents  Downloads  Library    Movies     Music      Pictures

# 環境変数を確認
$ echo $PATH
/usr/local/bin:/usr/bin:/bin:/usr/sbin:/sbin
```
              


              :::message
**確認完了**
上記のコマンドが実行できれば、ターミナルの基本的な操作は問題ありません。
:::

          ### チャレンジ: ターミナルの基本を確認しよう

tmux を学ぶ前に必要な基本的なターミナルコマンドを書いてください。カレントディレクトリの表示、ファイル一覧、環境変数の確認を行いましょう。

```tsx
# ターミナルの基本確認

# 1. 現在のディレクトリを表示

# 2. ホームディレクトリのファイル一覧

# 3. PATH 環境変数を表示

# 4. シェルの種類を確認
```

:::details 模範解答
```tsx
# ターミナルの基本確認

# 1. 現在のディレクトリを表示
pwd

# 2. ホームディレクトリのファイル一覧
ls ~

# 3. PATH 環境変数を表示
echo $PATH

# 4. シェルの種類を確認
echo $SHELL
```
:::
