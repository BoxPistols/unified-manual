# 
            実践的な MCP 連携
          
          
            Figma、Context7、Serena など、実際の開発で活用される主要MCPサーバの紹介。
          

        


        
          
          
            ## 
              
              デザイン → コード
            

            
            
              
                
                ### Figma MCP
                公式

                HTTP

              

              
                FigmaのデザインファイルからコードやUIコンポーネントを生成します。デザイン変数、レイアウト情報、Code Connect連携に対応。OAuth認証で接続。
              

              ```bash
# リモートサーバ（推奨・デスクトップアプリ不要）
$ claude mcp add --transport http figma https://mcp.figma.com/mcp

# 認証: セッション内で /mcp → figma → Authenticate
```
              
                
                  デザインコンテキスト取得

                  フレーム・変数・コンポーネント情報を抽出
                

                
                  Code Connect

                  既存コードベースのコンポーネントを再利用した出力
                

              

            

          


          
          
            ## 
              
              ドキュメント参照
            

            
            
              
                
                ### Context7
                @upstash/context7-mcp

              

              
                ライブラリのバージョン固有ドキュメントとコード例をソースから直接取得します。LLMの学習データが古くなる問題を解決し、最新APIのハルシネーションを防止します。
              

              ```bash
# セットアップ
$ claude mcp add --scope user context7 -- npx -y @upstash/context7-mcp

# 使い方: プロンプトに "use context7" を含めるだけ
> Next.js 15のApp Routerでページを作って (use context7)
```
              :::message
**APIキー**
APIキーなしでも動作しますが、レート制限が厳しいため context7.com/dashboard での無料取得を推奨します。
:::
            

          


          
          
            ## 
              
              コード理解・ナビゲーション
            

            
            
              
                
                ### Serena
                LSPベース

              

              
                Language Server Protocol（LSP）を活用し、30+言語に対してシンボルレベルのコード理解を提供します。ファイル全体を読まずにシンボル単位で精密にコードを操作可能。
              

              ```bash
# セットアップ（uvが必要）
$ claude mcp add serena -- uvx \
    --from git+https://github.com/oraios/serena \
    serena start-mcp-server --context claude-code --project "$(pwd)"
```
              
                
                  `find_symbol`
                  シンボルの定義を検索

                

                
                  `find_referencing_symbols`
                  参照箇所を全て検出

                

                
                  `insert_after_symbol`
                  シンボル前後にコード挿入

                

              

            

          


          
          
            ## 
              
              ブラウザ自動化
            

            
            
              
                
                ### Playwright MCP
                Microsoft公式

              

              
                アクセシビリティツリーを活用したブラウザ自動化。スクリーンショットベースではなく構造化データで動作するため、ビジョンモデル不要で高速・確定的に動作します。Chromium / Firefox / WebKit対応。
              

              ```bash
$ claude mcp add playwright -- npx @playwright/mcp@latest
```
            

          


          
          
            ## 
              
              監視・バックエンド連携
            
            
              
                
                  
                  #### Sentry MCP
                  HTTP

                

                エラー・イシュー検索、Seer AI分析、リリース管理にLLMから直接アクセス。

                `claude mcp add --transport http sentry https://mcp.sentry.dev/mcp`
              

              
                
                  
                  #### Supabase MCP
                  HTTP

                

                DB操作、マイグレーション、Edge Functions、ログ解析をAIから直接実行。

                `claude mcp add --transport http supabase https://mcp.supabase.com/mcp`
              

            

          


          
          
            ## 実践的なプロンプト例
            
              
                Figma MCP + Context7

                
                  &gt; この Figma URL のデザインを React コンポーネントとして実装して (use context7)

                  &gt; https://figma.com/design/xxxxx?node-id=1-2
                

              

              
                Sentry MCP

                
                  &gt; Sentry の未解決エラーを確認して、最も発生頻度の高い問題を修正して
                

              

              
                Playwright MCP

                
                  &gt; localhost:3000 のログインフォームをテストして、バリデーションが正しく動くか確認して
                

              

            

          


          
          
            ## HTTP MCPとOAuth認証
            
              Figma、Sentry、SupabaseなどのHTTPリモートサーバは、OAuth認証フローに対応しています。セッション内で `/mcp` コマンドからインタラクティブに認証を完了できます。
            

            :::message
**セキュリティ**
MCP経由の外部サービス呼び出しは、パーミッション設定の対象です。settings.json の allow/deny ルールで mcp__サーバ名__* パターンを使用して制御できます。
:::
          

          ### チャレンジ: MCP サーバをセットアップしよう

プロジェクトで使用する MCP サーバの登録コマンドを書いてください。Context7（ドキュメント参照）と Playwright（ブラウザ自動化）の2つを登録しましょう。

```tsx
# MCP サーバの登録コマンド

# 1. Context7（ドキュメント参照）をユーザースコープで登録

# 2. Playwright（ブラウザ自動化）をプロジェクトスコープで登録
```

:::details 模範解答
```tsx
# MCP サーバの登録コマンド

# 1. Context7（ドキュメント参照）をユーザースコープで登録
claude mcp add --scope user context7 -- npx -y @upstash/context7-mcp

# 2. Playwright（ブラウザ自動化）をプロジェクトスコープで登録
claude mcp add -s project playwright -- npx @playwright/mcp@latest
```
:::
