# 
            tmux.confの基本
          

          
            設定ファイルを作成して、tmuxを自分の好みにカスタマイズする第一歩を踏み出しましょう。
          

        


        
          
            ## 
              設定ファイルの場所
            
            
              tmuxの設定はホームディレクトリ直下の `.tmux.conf` というファイルに記述します。デフォルトではこのファイルは存在しないため、自分で作成する必要があります。
            

            ```bash
# 設定ファイルの作成（または開く）
$ touch ~/.tmux.conf
$ nano ~/.tmux.conf
```
          


          
            ## 
              まず設定すべき3つの基本
            
            
              まずは以下の3つの設定を追加することをお勧めします。これだけで使い勝手が劇的に向上します。
            


            
              
                ### 1. マウス操作を有効にする
                
                  ペインのサイズ調整やスクロール、ウィンドウ切り替えをマウスで行えるようになります。初心者に必須の設定です。
                

                ```bash
set -g mouse on
```
              


              
                ### 2. インデックスを1から始める
                
                  デフォルトではウィンドウ番号は0から始まりますが、キーボードの数字キーの並び（1が左端）に合わせて1から始める方が直感的です。
                

                ```bash
set -g base-index 1
setw -g pane-base-index 1
```
              


              
                ### 3. 256色表示を有効にする
                
                  ターミナルの色が正しく表示されるようにします。Vimなどで色が崩れるのを防ぎます。
                

                ```bash
set -g default-terminal "screen-256color"
```
              

            

          


          
            ## 
              設定の反映
            
            
              ファイルを保存しただけでは設定は反映されません。以下のコマンドを実行して設定を再読み込みするか、tmuxを一度終了して再起動する必要があります。
            

            
            :::message alert
**注意**
tmuxセッション内で実行してください。
:::

            ```bash
# 設定ファイルの再読み込み
$ tmux source ~/.tmux.conf
```
          


          
            ## 
              まとめ: 最初の.tmux.conf
            
            
              ここまでの内容をまとめたファイルです。これをコピーして `~/.tmux.conf` に貼り付けてください。
            

            ```bash
# マウス操作を有効にする
set -g mouse on

# インデックスを1から始める
set -g base-index 1
setw -g pane-base-index 1

# 256色端末を使用する
set -g default-terminal "screen-256color"
```
          


          ### チャレンジ: tmux.conf を書いてみよう

プリフィックスキーを Ctrl+a に変更し、マウス操作を有効にし、ウィンドウ番号を1から始める設定を書いてください。

```tsx
# tmux.conf を完成させてください

# プリフィックスキーを Ctrl+a に変更

# マウス操作を有効にする

# ウィンドウ番号を1から始める
```

:::details 模範解答
```tsx
# プリフィックスキーを Ctrl+a に変更
unbind C-b
set -g prefix C-a
bind C-a send-prefix

# マウス操作を有効にする
set -g mouse on

# ウィンドウ番号を1から始める
set -g base-index 1
setw -g pane-base-index 1
```
:::
