          # 
            GitHub CLI を導入しよう
          

          
            GitHub CLI（gh）は、ターミナルから GitHub を操作できる強力なツールです。リポジトリの作成、プルリクエストの作成・マージ、Issue の管理など、Web ブラウザを使わずにすべてを実行できます。開発効率が飛躍的に向上します。
          

        


        

        {selectedOS === 'windows' && (
          
            
              ## 
                GitHub CLI のインストール
              

              
                WSL2 のターミナルで以下のコマンドを実行して、GitHub CLI をインストールします。
              


              ```bash
sudo apt update
sudo apt install gh
```

              :::message
**インストール確認**
以下のコマンドでバージョンを確認できます：gh --version
:::
            


            
              ## 
                GitHub CLI で認証
              

              
                GitHub CLI を使用する前に、GitHub アカウントで認証します。以下のコマンドを実行してください。
              


              ```bash
gh auth login
```

              
                ### 対話的なセットアップ
                以下の質問に答えていきます：

                ```bash
? What is your preferred protocol for Git operations? [Use arrows to move, type to filter]
> HTTPS
  SSH

? Authenticate with your GitHub credentials? [Y/n]
Y

? How would you like to authenticate GitHub CLI?
> Login with a web browser
  Paste an authentication token
```
              


              :::message
**認証完了**
ブラウザが開き、GitHub のログイン画面が表示されます。指示に従ってログインすれば、認証が完了します。
:::
            


            
              ## 
                GitHub CLI の基本コマンド
              

              
                
                  ### 
                    1. リポジトリを作成
                  
                  
                    ターミナルから新しいリポジトリを作成できます。
                  

                  ```bash
gh repo create my-new-project --public --source=. --remote=origin --push
```
                  
                    --public: 公開リポジトリ（--private で非公開）

                    --source=.: 現在のディレクトリをソースに

                    --remote=origin: リモート名を origin に

                    --push: 自動的に push
                  

                


                
                  ### 
                    2. リポジトリをクローン
                  
                  
                    GitHub のリポジトリをクローンします。
                  

                  ```bash
gh repo clone username/repository
```
                


                
                  ### 
                    3. プルリクエストを作成
                  
                  
                    ブランチからプルリクエストを作成します。
                  

                  ```bash
gh pr create --title "新機能を追加" --body "この PR では新しい機能を追加しました。"
```
                


                
                  ### 
                    4. プルリクエストをマージ
                  
                  
                    プルリクエストをマージします。
                  

                  ```bash
gh pr merge
