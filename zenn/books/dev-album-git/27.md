            ## 
              Gemini CLI
            
            ターミナル型

              Claude Code と同じ感覚で使えて、**1日1,000リクエストまで無料**という太っ腹な無料枠が魅力です。
            

            :::message
**無料枠が大きい**
Google アカウントがあれば追加登録不要。毎日1,000リクエストまで無料なので、
              Claude Code のクレジットが尽きた時の切り替え先として最適です。
:::
