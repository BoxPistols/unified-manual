          # 
            Linux/Ubuntu 基礎を学ぼう
          
          
            ターミナルで使う基本的なコマンドとファイルシステムの仕組みを理解します。
          

        

      


      
      
        
        
          ## 
            ターミナルの基本
          
          
            
              ターミナルは、コマンド（テキスト）でコンピュータに指示を出すツールです。GUI（マウスで操作する画面）と異なり、キーボードだけで操作します。
            

            
              
                username@hostname:~$
              

            

            
              このような表示が出たら、コマンドを入力する準備ができています。`$` の後にコマンドを入力して Enter キーを押します。
            

          

        


        
        
          ## 
            ファイルシステムの理解
          
          
            
              Linux のファイルシステムは、ツリー構造になっています。最上位のディレクトリを「ルート」（`/`）と呼びます。
            

            
              / (ルート)

              ├── home/ (ユーザーのホームディレクトリ)

              └── username/

              ├── var/ (ログファイルなど)

              ├── etc/ (設定ファイル)

              └── usr/ (プログラムやライブラリ)

            

            :::message
ホームディレクトリ（~）は、ユーザーの個人用ディレクトリです。ここにファイルやプロジェクトを保存します。
:::
          

        


        
        
          ## 
            よく使うコマンド
          

          
          
            ### pwd - 現在のディレクトリを表示
            ```bash
pwd
```
            
              現在いるディレクトリのパスを表示します。「どこにいるのか」を確認するときに使います。
            

          


          
          
            ### ls - ファイル一覧を表示
            ```bash
ls              # 現在のディレクトリのファイル一覧
ls -la          # 詳細情報を表示（隠しファイルも含む）
ls -lh          # ファイルサイズを人間が読みやすい形式で表示
```
            
              ディレクトリ内のファイルやフォルダを一覧表示します。
            

          


          
          
            ### cd - ディレクトリを移動
            ```bash
cd Documents    # Documents フォルダに移動
cd ..           # 親ディレクトリに移動
cd ~            # ホームディレクトリに移動
cd /            # ルートディレクトリに移動
```
            
              別のディレクトリに移動します。`..` は親ディレクトリを表します。
            

          


          
          
            ### mkdir - 新しいディレクトリを作成
            ```bash
mkdir my-project        # my-project フォルダを作成
mkdir -p a/b/c          # ネストされたフォルダを作成
```
            
              新しいフォルダを作成します。`-p` オプションで、親ディレクトリも同時に作成できます。
            

          


          
          
            ### touch - 空のファイルを作成
            ```bash
touch README.md         # README.md ファイルを作成
touch index.html        # index.html ファイルを作成
```
            
              空のファイルを作成します。ファイルの内容を後で追加できます。
            

          


          
          
            ### cat - ファイルの内容を表示
            ```bash
cat README.md           # README.md の内容を表示
cat file1.txt file2.txt # 複数ファイルの内容を表示
```
            
              ファイルの内容をターミナルに表示します。
            

          


          
          
            ### nano - テキストエディタ
            ```bash
nano README.md          # README.md をエディタで開く
```
            
              ターミナル内でテキストファイルを編集できます。編集後は Ctrl+X で終了し、Y を入力して保存します。
            

          


          
          
            ### rm - ファイルを削除
            ```bash
rm file.txt             # ファイルを削除
rm -r folder/           # フォルダとその中身を削除
```
            :::message alert
削除したファイルはゴミ箱に入らず、完全に削除されます。慎重に使用してください。
:::
          


          
          
            ### cp - ファイルをコピー
            ```bash
cp file.txt file_copy.txt       # ファイルをコピー
cp -r folder/ folder_copy/      # フォルダをコピー
```
            
              ファイルやフォルダをコピーします。
            

          


          
          
            ### mv - ファイルを移動・名前変更
            ```bash
mv file.txt new_name.txt       # ファイルを名前変更
mv file.txt Documents/          # ファイルを移動
mv old_folder/ new_folder/      # フォルダを移動・名前変更
```
            
              ファイルを移動したり、名前を変更したりします。
            

          

        


        
        
          ## 
            ユーザーと権限
          
          
            
              Linux では、ファイルやコマンドに対して「誰が何をできるか」という権限が設定されています。
            


            
              ### sudo - 管理者権限で実行
              ```bash
sudo apt update         # パッケージリストを更新（管理者権限が必要）
sudo apt install git    # Git をインストール
```
              
                `sudo` を付けると、コマンドを管理者権限で実行できます。パスワードの入力が求められることがあります。
              

            


            
              ### chmod - ファイルの権限を変更
              ```bash
chmod +x script.sh      # ファイルを実行可能にする
chmod 755 file.txt      # 権限を数値で指定
```
              
                ファイルの読み取り、書き込み、実行権限を変更します。
              

            

          

        


        
        
          ## 
            実践練習
          
          
            
              以下の手順で、実際にコマンドを使ってみましょう。
            

            
              - 
                `mkdir my-project` でフォルダを作成
              
              - 
                `cd my-project` でそのフォルダに移動
              
              - 
                `touch README.md` でファイルを作成
              
              - 
                `nano README.md` でファイルを編集
              
              - 
                `cat README.md` でファイルの内容を確認
              
            

            :::message
これらのコマンドが使えるようになれば、Linux の基本操作はマスターです！
:::
          

        


        
        
          ### チャレンジ: Linux 基本コマンドで プロジェクトを作成してみよう

フォルダを作成し、ファイルを追加して内容を確認するまでのコマンドを書いてください。

```tsx
# 1. my-project フォルダを作成

# 2. my-project フォルダに移動

# 3. README.md ファイルを作成

# 4. README.md にテキストを書き込む

# 5. README.md の内容を表示

```

:::details 模範解答
```tsx
# 1. my-project フォルダを作成
mkdir my-project

# 2. my-project フォルダに移動
cd my-project

# 3. README.md ファイルを作成
touch README.md

# 4. README.md にテキストを書き込む
echo "# My Project" > README.md

# 5. README.md の内容を表示
cat README.md
```
:::
