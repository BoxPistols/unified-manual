          # 
            AI コーディング環境の全体像
          
          
            無料〜月1,000円以下で、AIと一緒にコーディングできる環境を構築します。
            複数のツールを組み合わせて、トークン切れでも作業が止まらない体制を作りましょう。
          

        

      


      
      

        
        
          ## 
            AI コーディングツールって何？
          
          
            
              AI コーディングツールとは、**AIに話しかけるだけでコードを書いたり、修正したり、質問に答えてもらえる**ツールのことです。
            

            
              例えば、こんなことができます:
            

            
              
                「ボタンの色を赤に変えて」

                → AI がコードを見つけて自動で修正

              

              
                「このエラーの原因は？」

                → AI がコードを分析して原因を説明

              

              
                「最新のコードを取得して」

                → AI が git pull を実行してくれる

              

              
                「ログインページを作って」

                → AI がファイルを作成してコードを生成

              

            

            :::message
**コマンドを覚えなくてOK**
Git コマンドやプログラミング言語の文法を完璧に覚えていなくても、
              日本語で指示するだけで AI が代わりにやってくれます。
              コーディングに参加するハードルが大幅に下がります。
:::
          

        


        
        
          ## 
            ツールは3タイプある
          
          
            
              
                
                  
                

                
                  ### ターミナル型
                  コマンドラインで AI と対話する

                

              

              
                ターミナル（黒い画面）で AI に話しかけるタイプ。シンプルで軽量。
                既存のエディタと組み合わせて使います。
              

              
                Claude Code

                Gemini CLI

                Warp（AI搭載ターミナル）

              

            


            
              
                
                  
                

                
                  ### エディタ拡張型
                  普段のエディタに AI 機能を追加する

                

              

              
                VS Code や Cursor などのエディタ内で AI と対話するタイプ。
                コードを見ながら指示できるので直感的。
              

              
                Cursor（AI搭載エディタ）

                Cline（VS Code 拡張）

              

            


            
              
                
                  
                

                
                  ### クラウド IDE 型
                  ブラウザで完結する AI 開発環境

                

              

              
                ブラウザ上で動く開発環境。インストール不要で始められるのが利点。
                複数の AI エージェントを同時に動かせるものもあります。
              

              
                Google Antigravity
