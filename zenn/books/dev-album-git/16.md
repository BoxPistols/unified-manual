          ## 
            App.js を編集してデザイン変更
          

          
            
            
              
                
                  1
                

                ### 
                  App.js を開く
                
              

              
                Cursor の左パネルで、「src/App.js」をクリックして開いてください。
              

            


            
            
              
                
                  2
                

                ### 
                  テキストを変更
                
              

              
                App.js の内容を以下のように編集してください。
              


              ```jsx
import logo from './logo.svg';
import './App.css';

function App() {
  return (
    
      
        <img src={logo}  alt="logo" />
        
          Welcome to My React App!
        

        
          I'm learning React and Git!
        

        <a
          
          href="https://reactjs.org"
          target="_blank"
          rel="noopener noreferrer"
        >
          Learn React
        </a>
      

    

  );
}

export default App;
```

              
                編集内容：
              

              
                - テキストを「Welcome to My React App!」に変更
                - 新しいメッセージを追加
                - テキストの色をエメラルドグリーン（#10B981）に変更
              


              :::message
ファイルを保存（Cmd + S / Ctrl + S）すると、ブラウザが自動的に更新されます。
:::
            


            
            
              
                
                  3
                

                ### 
                  ブラウザで変更を確認
                
              

              
                ブラウザで http://localhost:3000 を確認して、変更が反映されているか確認してください。
              


              :::message
テキストと色が変更されていれば、デザイン変更成功です！
:::

                ### 
                  GitHub に Push してセルフマージ
                
              

              
                ブランチを GitHub に Push してください。
              


              

              
                GitHub のリポジトリページを開くと、「Create Pull Request」というボタンが表示されます。クリックして、Pull Request を作成してください。
              


              
                - GitHub のリポジトリページを開く
                - 「Create Pull Request」をクリック
                - タイトルと説明を入力（例：「Update welcome message and styling」）
                - 「Create Pull Request」をクリック
                - Pull Request ページで、変更内容を確認
                - 「Merge pull request」をクリックしてマージ

          ### チャレンジ: React プロジェクトの Git ワークフローを書いてみよう

ブランチを作成し、ファイルをステージング・コミットして、GitHub に Push するまでの一連のコマンドを書いてください。

```tsx
# 1. 新しいブランチを作成して切り替え

# 2. 変更したファイルをステージング

# 3. コミット

# 4. GitHub に Push

```

:::details 模範解答
```tsx
# 1. 新しいブランチを作成して切り替え
git checkout -b feature/update-welcome-message

# 2. 変更したファイルをステージング
git add src/App.js

# 3. コミット
git commit -m "Update welcome message and styling"

# 4. GitHub に Push
git push -u origin feature/update-welcome-message
```
