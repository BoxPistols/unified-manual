        # React Router
        
          複数ページのアプリを作るための React Router を学びましょう。URL に応じたページの切り替え、
          ページ間のナビゲーション、動的なルーティングを実装します。
        

        :::message
**なぜ今これを学ぶのか**
React はデフォルトではシングルページアプリケーション（SPA）で、URL の切り替え機能がありません。
            React Router を使えば、/about や /products/123 のように
            URL に応じて異なるコンポーネントを表示できます。
            実際のアプリ開発には必須の知識です。
:::

            :::message alert
**a タグを使わない理由**
通常の &lt;a href="/about"&gt; タグを使うと、ページ全体がリロードされてしまいます。
                React Router の Link を使えば、JavaScript で画面の一部だけを差し替えるため、
                高速でスムーズなページ遷移が実現できます。これが SPA の大きなメリットです。
:::
